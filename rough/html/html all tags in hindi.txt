Tag	Description
<!doctype>	Document Type को Define करने के लिए
<–…–>	Comment Tag को source code के बारे में describe करने के लिए किया जाता है.
<html>	HTML document को Define करने के लिए
<head>	HTML document के header section को परिभाषित करता है.
<title>	HTML document का Title Define करने के लिए.
<body>	HTML document की Body को Define करता है.
<style>	HTML में Style Information को Define किया जाता है.
<H1>to<H6>	HTML में H1 से H6 Level तक Headings बनाने के लिए.
<p>	Paragraph बनाने के लिए किया जाता है.
<hr>	इस tag से Horizontal Line को Define किया जाता है.
<br>	(Break) Single Line Break देने के लिए उपयोग किया जाता है.
<span>	Document का section को Define करने के लिए.
<div>	(Division) Document में अलग-अलग Sections को Define करने के लिए.
<a>	Anchor Element का उपयोग Hyperlink बनाने के लिए किया जाता है.
<link>	External Document को HTML Document से link करने के लिए.
<meta>	Document का Meta Data Define करने के लिए.
<img>	Image को Define किया जाता है.
<map>	Image-Map को Define करने के लिए.
<area>	Image-Map में एक विशेष Area को Define करने के लिए.
<base>	Related URLs के लिए एक Base URL Define करने के लिए.
<script>	Client side Script को Define किया जाता है.
<nonscript>	इसके द्वारा Alternet Content को लिखा जाता है, जो Script को Support नहीं करते है.
<frame>	Document में Frames को Define किया जाता है.
<frameset>	Frameset को Define किया जाता है.
<noframes>	इसके द्वारा Alternet Content को लिखा जाता है, जो Frames को Support नहीं करते है.
<iframe>	Inline Frame को Define किया जाता है.

<i>	Text को Italic करने के लिए.

<pre>	Preformatted Text को Define किया जाता है.
